# the-beer-baron

## Installation

This project uses Python 3.6 for the Django server, and a virtual environment to manage dependencies.

### Running client

```
cd client
yarn
yarn start
```
Open xcode project in `/ios`.

### Running backend
```
cd backend
yarn
yarn start
```

In terminal, spin up `mongodb`:
```
mongod
```
Then, in another terminal:
```
mongo
```
